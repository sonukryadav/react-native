### ExpoApp
- It is the pure Expo app.

- To create a project:
- npx create-expo-app <ProjectName>

- To run a project:
- npx expo start



### ReactNativeApp
- It is the bare react native project

- To create a project:
- npx react-native@latest init <ProjectName>

- To run a project:
- npx react-native run-android
